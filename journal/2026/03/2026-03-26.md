# 2026-03-26 每日 AI 技術摘要

## 今日重點

Anthropic 在 3 月 24 日正式宣布 Claude 電腦操控（Computer Use）功能登陸 macOS，Claude 現在可以開啟應用程式、操作瀏覽器、填寫試算表，標誌 AI Agent 自主化進入新階段。OpenAI 則持續推進 GPT-5.4 系列的鋪開，mini 版本已向免費用戶開放。

---

## Anthropic / Claude

- **Claude Computer Use 登陸 macOS**：Claude 現可操控桌面應用程式、瀏覽器與試算表，完成跨 app 的複合任務。新功能「Dispatch」讓使用者從手機或桌機指派 Agent 任務並持續對話。
  - 來源：[CNBC](https://www.cnbc.com/2026/03/24/anthropic-claude-ai-agent-use-computer-finish-tasks.html) ｜ [Dataconomy](https://dataconomy.com/2026/03/24/anthropic-adds-computer-use-to-claude-on-macos/)

- **Claude Code 持續更新**：新增 `--channels` research preview（MCP 訊息推送）、bare mode（適合 scripted `-p` 呼叫）、channel-based permission relay；修復 OAuth、voice mode、session、plugin 及 Windows 相關問題。
  - 來源：[Claude Code Changelog](https://code.claude.com/docs/en/changelog)

- **TechCrunch 深度報導**：Anthropic 為 Claude Code 開放更多系統控制能力，但同步強化安全護欄——對應「更多自主性 × 更嚴格限制」的雙軌策略。
  - 來源：[TechCrunch](https://techcrunch.com/2026/03/24/anthropic-hands-claude-code-more-control-but-keeps-it-on-a-leash/)

- **API 新功能**：Extended Thinking 新增 `thinking.display: "omitted"` 設定，可省略回應中的思考內容以加速 streaming，但保留 signature 維持多輪對話一致性。Web Search 工具與程式化 tool calling 正式 GA。
  - 來源：[Anthropic Release Notes](https://platform.claude.com/docs/en/release-notes/overview)

## OpenAI

- **GPT-5.4 mini 開始向免費用戶推送**：Free 和 Go 用戶可透過「Thinking」功能使用，其餘用戶作為 GPT-5.4 Thinking 的 rate limit fallback。
  - 來源：[ChatGPT Release Notes](https://help.openai.com/en/articles/6825453-chatgpt-release-notes)

- **GPT-5.4 系列回顧（3/5 發布）**：GPT-5.4 整合推理、coding 與 agentic 工作流，是 OpenAI 首個具備原生電腦操控能力的通用模型。GPT-5.4 Pro 針對複雜工作負載提供最高效能。
  - 來源：[TechCrunch](https://techcrunch.com/2026/03/05/openai-launches-gpt-5-4-with-pro-and-thinking-versions/) ｜ [Help Net Security](https://www.helpnetsecurity.com/2026/03/06/openai-chatgpt-gpt%E2%80%915-4-model-release/)

## GitHub Copilot / 其他工具

- **Copilot 資料使用政策異動（3/25 公告）**：自 2026/04/24 起，Free、Pro、Pro+ 用戶的互動資料將用於訓練 AI 模型（可至隱私設定關閉）；Business 和 Enterprise 用戶不受影響。
  - 來源：[GitHub Blog](https://github.blog/news-insights/company-news/updates-to-github-copilot-interaction-data-usage-policy/)

- **Copilot for Students 方案調整（3/12 起）**：學生方案現改為獨立 Copilot Student Plan，GPT-5.4 與 Claude Opus/Sonnet 等 premium 模型不再提供手動選取，但 Auto mode 仍可自動選用適合模型。
  - 來源：[GitHub Changelog](https://github.blog/changelog/2026-03-13-updates-to-github-copilot-for-students/)

- **Copilot CLI v1.0.10（3/20）**：減少大型檔案 memory 使用量，新增 `/undo` 指令與 experimental multi-session 支援。
  - 來源：[Copilot CLI Releases](https://github.com/github/copilot-cli/releases)

- **Copilot Code Review 達 6000 萬次**：自 2025/04 推出至今，成長 10 倍。

## 對 UI 前端開發的影響

1. **Claude Computer Use × Figma／設計工具**：Claude 現在可以操控桌面應用，代表未來可能直接在 Figma 上進行設計操作，或自動完成 design-to-code 的最後一哩路。Henry 可以關注 Claude + Figma MCP 的組合玩法。

2. **AI 前端生成工具已形成生態**：Vercel v0、Kombai、Windframe 等工具以 React + Tailwind CSS 為核心棧。搭配 Claude Code 的新 MCP channel 功能，可以想像 Agent 直接接收設計 prompt 並輸出可部署的元件。

3. **GPT-5.4 的 Computer Use 能力**：OpenAI 與 Anthropic 雙雙進入「AI 直接操控電腦」的階段，對自動化前端測試（如自動跑 E2E、截圖比對）有潛在應用。

4. **Copilot 學生方案限縮 premium 模型**：如果你在教育情境使用 Copilot，需留意 Claude Sonnet 等模型可能無法手動切換，但 Auto mode 仍會嘗試選用最佳模型。
