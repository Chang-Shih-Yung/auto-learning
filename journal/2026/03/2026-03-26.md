# 2026-03-26 學習日誌

## 今日重點

Anthropic 持續強化 Claude Code 的自主能力：3 月份連續更新（v2.1.63 → v2.1.76+）帶來語音操作、`/loop` 循環任務、1M token context window 和 Opus 4.6 設為預設模型。與此同時，OpenAI GPT-5.4 系列的 mini 版本向免費用戶開放，Google 也推出 Gemini 3.1 Flash-Lite，AI 模型競爭進入多強並立的新格局。

## Anthropic / Claude

- **Claude Code 三月密集更新（v2.1.63 → v2.1.76+）**：
  - **Voice Mode（Push-to-Talk）**：以 `/voice` 啟動，按住空白鍵說話、放開發送；正在向全用戶漸進推出
  - **`/loop` 循環任務**：支援設定固定間隔重複執行 prompt 或 slash command，可搭配排程任務使用
  - **1M Token Context Window**：Claude Code 擴展至 100 萬 token 上下文
  - **Opus 4.6 設為預設**：Bedrock、Vertex、Microsoft Foundry 上的預設 Opus 模型由 4.1 升至 4.6
  - **`--bare` flag**：scripted `-p` 呼叫模式，跳過 hooks、LSP、plugin sync，需搭配 `ANTHROPIC_API_KEY`
  - **`--channels` permission relay**：Channel server 可將工具核准 prompt 轉發至手機，實現遠端審批
  - 來源：[Claude Code Changelog](https://code.claude.com/docs/en/changelog) ｜ [Releasebot](https://releasebot.io/updates/anthropic/claude-code)

- **Claude Computer Use 登陸 macOS（3/24）**：Claude 可操控桌面應用程式、瀏覽器與試算表。新功能「Dispatch」讓用戶從手機或桌機指派 Agent 任務並持續對話。
  - 來源：[CNBC](https://www.cnbc.com/2026/03/24/anthropic-claude-ai-agent-use-computer-finish-tasks.html) ｜ [Dataconomy](https://dataconomy.com/2026/03/24/anthropic-adds-computer-use-to-claude-on-macos/)

- **TechCrunch：Claude Code 更多控制 × 更嚴護欄**：Anthropic 賦予 Claude Code 更多系統層級控制能力，同步強化安全限制，體現「自主性 ↑、安全邊界也 ↑」的雙軌策略。
  - 來源：[TechCrunch](https://techcrunch.com/2026/03/24/anthropic-hands-claude-code-more-control-but-keeps-it-on-a-leash/)

- **API 新功能**：Extended Thinking 新增 `thinking.display: "omitted"`，可省略回應中的思考內容加速 streaming，保留 signature 維持多輪對話一致性。Web Search 工具與 programmatic tool calling 正式 GA。
  - 來源：[Anthropic Release Notes](https://platform.claude.com/docs/en/release-notes/overview)

## OpenAI

- **GPT-5.4 mini 向 Free 用戶推送**：Free 和 Go 用戶可透過「Thinking」功能使用；其他用戶作為 GPT-5.4 Thinking 的 rate limit fallback。
  - 來源：[ChatGPT Release Notes](https://help.openai.com/en/articles/6825453-chatgpt-release-notes)

- **GPT-5.4 & GPT-5.4 Pro（3/5 發布）**：整合推理、coding 與 agentic 工作流，支援 1M token context window；GPT-5.4 是 OpenAI 首個具備原生電腦操控能力的通用模型。GPT-5.4 Pro 的「tool search」功能可在 tool-heavy 環境減少 47% token 使用量。
  - 來源：[TechCrunch](https://techcrunch.com/2026/03/05/openai-launches-gpt-5-4-with-pro-and-thinking-versions/)

- **GPT-5.1 系列自 3/11 起全面下線**，包含 Instant、Thinking、Pro 變體。

## 對 UI 前端開發的影響

1. **Claude Code Voice Mode → 聲控前端開發**：`/voice` 功能讓你可以說出「幫我把這個 button 改成 Tailwind 的 rounded-full」，邊看設計稿邊口述修改，適合 design review 場景。

2. **`/loop` 指令 → 自動化重複任務**：可設定每 N 分鐘執行一次 prompt，例如定期跑 UI regression check 或自動產生元件截圖。Claude_auto 的排程任務就是這個模式的延伸。

3. **Claude Computer Use × Figma**：Claude 現在可操控桌面應用，代表未來有機會直接在 Figma 進行設計操作，或自動完成 design-to-code 最後一哩。值得關注 Claude + Figma MCP 的組合玩法。

4. **1M Token Context**：可以把整個前端 repo 的核心檔案一次餵進去，讓 Claude Code 理解全局再提出改動建議，不再需要手動挑選 context。

5. **AI 前端生成工具生態成熟**：Vercel v0、Kombai、Windframe 以 React + Tailwind CSS 為核心棧，搭配 Claude Code 的 MCP channel 功能，Agent 直接接收設計 prompt 並輸出可部署元件的工作流已觸手可及。
