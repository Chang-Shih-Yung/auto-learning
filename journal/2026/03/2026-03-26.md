### ARC-AGI-3：互動式智能基準登場

**來源：** [arcprize.org/arc-agi/3](https://arcprize.org/arc-agi/3) · HN 368 pts

ARC-AGI-3 是第三代 AGI 評測基準，核心突破在於「**互動性**」——不再是靜態解題，而是要求 AI agent 在陌生環境中自主探索、即時建立目標、持續更新世界模型。

**與前代的關鍵差異：**
- 評估「技能習得效率」隨時間的變化，而非單次答題正確率
- 測試長時程規劃（long-horizon planning）搭配稀疏回饋（sparse feedback）
- 所有環境 100% 可由人類解決，附 replay 功能供行為透明審查

滿分定義：「AI agent 能以與人類同等效率通關每一個環境。」只要 AI 與人類在學習效率上仍有落差，ARC 就認為 AGI 尚未實現。這把評估維度從「會不會」拉到「學得快不快」，更接近人類智能的本質。

### "Disregard That" 攻擊：Prompt Injection 的結構性困境

**來源：** [calpaterson.com/disregard.html](https://calpaterson.com/disregard.html) · HN 66 pts

文章剖析 prompt injection 的根本問題：當 LLM 的 context window 同時包含「受信任的系統 prompt」與「不受信任的使用者輸入」，攻擊者只需在輸入裡加入「忽略前面指令」即可劫持系統行為。

**關鍵洞察：**
- 加防禦指令是軍備競賽，無法根治（攻擊者升級 prompt 即可繞過）
- 不受信任資料進入 context 的路徑很多：API、web search、文件、使用者輸入
- 有效防禦：**限制不受信任內容進入 context**，或要求人工審查，或讓 LLM 輸出可審查的程式碼而非直接執行動作

核心比喻：「把 context window 分享給未審查的資料來源，等同於把系統控制權拱手相讓。」

### Axe：Unix 哲學的 12MB AI Agent CLI

**來源：** [github.com/jrswab/axe](https://github.com/jrswab/axe) · HN 227 pts

Axe 是一個 12MB 的 CLI binary，用 TOML 定義 agent，讓每個 AI agent 像 Unix 程式一樣：**小、專注、可組合**。

**架構概念：**
- 每個 agent 有獨立的 system prompt、model 選擇、技能文件
- 支援 pipe 輸入輸出，可串接 git hook、cron job、grep 等工具
- Multi-provider（Anthropic、OpenAI、Ollama、AWS Bedrock）
- Sub-agent 委派、persistent memory、MCP server 整合內建

**Unix 哲學 × AI：** 不做全能聊天機器人，而是做很多個單一功能的小 agent，透過 pipe 串接。行為更可預測、可測試、可除錯——而且攻擊面更小（對應上一篇的 prompt injection 問題）。

## 今日總結

今天的內容圍繞著一個核心主題：**AI agent 的能力邊界與安全邊界正在同時被推進**。

**三個層面的平行進展：**

1. **評估維度升級**（ARC-AGI-3）：從「會不會」到「學得快不快」，benchmark 開始追趕 agent 能力的本質。

2. **攻擊面擴大**（Disregard That + Axe）：agent 越強大、整合越深，context window 的信任邊界問題越急迫。Axe 的 Unix 哲學——讓每個 agent 小且專注——其實也是一種防禦策略：攻擊面小的 agent 更容易審計。

3. **工具鏈成熟**（Claude Code v2.1.84、SDK v0.86.0）：企業部署能力與 agent 記憶架構同步補完，說明 AI agent 從「實驗性」走向「生產就緒」的基礎設施正在快速補齊。

**對 Henry 最直接的應用點：**
- CI/CD agent 設計時，把 prompt injection trust boundary 納入架構考量（對應 Disregard That 的防禦框架）
- Claude_auto 的 memory 系統可關注 SDK v0.86.0 的 filesystem memory tools，評估是否值得遷移
- ARC-AGI-3 的評估框架（稀疏回饋 + 長時程規劃）可借用來設計自己的 agent evaluation 標準
