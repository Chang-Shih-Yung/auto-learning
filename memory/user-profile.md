## 重要專案

### EQS 全球報價系統（東元機電）
- 角色：SD + UI/UX，第一次主導系統級專案
- 問題：100+ 表單欄位無卡控、舊架構用 ng-if 導致 DOM 銷毀重建，操作中斷率 100%
- 解法：前端流程卡控（必填/格式/金額門檻）、共用主容器 + 權限模型架構 A/B 測試
- 成果：企業內控強化、跨地域統一介面規格

### MES 智慧製造系統（東元集團）
- 角色：前端主導 + 協同三名開發，走完完整五個階段生命週期
- 問題：工時紙本抄寫、設備狀態無法即時追蹤
- 解法：PC + PDA 雙端即時報工、戰情室看板（forkJoin 並行 API、hover 按需打 API）
- 成果：**上工操作從 1分20秒 → 20秒，效率提升 75%**、錯誤率降 33%

### Nexus 企業級 UI 元件庫（自主主導）
- Figma → JSON → CSS Token → Angular Component 完整鏈路
- AI Agent 自動讀取 Figma node-id 拼裝 UI Block，60% 元件 AI 生成
- CLI 工具：`npx @nexus/cli add button`
- CI/CD 整合 SAST 白箱資安
